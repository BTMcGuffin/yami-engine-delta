Class: Layer

YED.Tilemap. Layer

new YED.Tilemap.Layer(data, tilesets, tileWidth, tileHeight)

Tilemap/Layer.js, line 21

The Layer object extends Sprite object in RMMV framework, is created by YED.Tilemap.Core to render a layer in tilemap

A Layer can be created and render by simply creating a new Layer object and add into a PIXI.DisplayObjectContainer object:

var layer = new YED.Tilemap.Layer(data, tilesets, tileWidth, tileHeight);
scene.addChild(layer);
