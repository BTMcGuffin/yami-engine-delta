Class: Data

YED.Tilemap. Data

new Data()

The Data object is created by YED.Tilemap.Core to load and contain tilemap data from json file

To make sure the scene is ready, loading method should be called manually:

var data = new YED.Tilemap.Data();
data.loadMapFile();
