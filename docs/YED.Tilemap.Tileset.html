Class: Tileset

YED.Tilemap. Tileset

new Tileset(data)

The Tileset objects are created by Tilemap.Core to load tileset textures and return tile blocks when needed

A Tileset object is created with a tileset data gets from Tilemap.Data#tilesets, for example:

var tilesetData = data.tilesets[0];
var tileset = new YED.Tilemap.Tileset(tilesetData);
